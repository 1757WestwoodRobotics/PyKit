Welcome to PyKit!

PyKit is a Pure-Python logging, telementary, and replay framework developed by `Team 1757 <https://whsrobotics.org>`_ for
use on Python robots. PyKit enables replaying log files to re-simulate robot code based on real data.

* `GitHub Repository <https://github.com/1757WestwoodRobotics/PyKit>`_
* `PyPI Page <https://pypi.org/project/pykit/>`_
* `Documentation <https://pykit.readthedocs.io/>`_
